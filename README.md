# Hand-energy
Base DP or QC power to Create heat !



### 主控制核心板

- 接口：Type-c

- 芯片：at32f415的 7mm超小封装
- 屏幕：一个LCD的1.3寸16位全彩屏
- 温度探测：DS18B20检测100℃以内温度
- LED：电源灯、功率灯、RGB
- 按键：自定义...

### 发热板

采用铝基板发热，理论依据是欧姆定律，将铜线按照希尔伯特曲线组成电阻。
